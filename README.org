* Dotfiles
My dotfiles.

** Usage
First, clone the repository with:

#+begin_src sh
brew tap aiyazmostofa/dotfiles git@github.com:aiyazmostofa/dotfiles.git
#+end_src

Then, enter the repository using:

#+begin_src sh
cd "$(brew --repository aiyazmostofa/dotfiles)"
#+end_src

Run =setup.sh= to complete the initial setup.
To sync the dotfiles, run =sync.sh=.

If you are not using a UBlue machine, then ensure that you have =brew= and =fish= installed first.
