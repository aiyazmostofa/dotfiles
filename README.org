* Dotfiles
My dotfiles.
Should work on Bluefin DX, Bluefin Dakota, and Bazzite DX.

** Usage
First, clone the repository with:

#+begin_src sh
brew tap aiyazmostofa/dotfiles git@github.com:aiyazmostofa/dotfiles.git
#+end_src

Then, enter the repository using:

#+begin_src sh
cd "$(brew --repository aiyazmostofa/dotfiles)"
#+end_src

Run =sync.sh= to complete the initial setup.
If something fails, keep retrying until it succeeds (very scuffed).
