* Dotfiles
My (NixOS) dotfiles.

** Setup
1. Clone this repo
   + You can/should clone this somewhere in your home folder
2. Edit the =configuration.nix= to your liking (described in the Configuration section)
3. Run =just update= to complete the setup
   + Run =nix-shell -p just= if you don't have =just= installed

** Usage
You can rebuild the configuration using the =just= command.
When you want to update channels, edit the =Justfile= accordingly and run =just update=.
You can format the Nix files with =just format=.

*** Features
- GNOME
- Preconfigured Emacs + Fish + Ghostty
  + For personal enjoyment, not meant for external use
- Direnv
- Distrobox/Podman
- Docker
- QEMU + KVM + libvirtd
  + virt-manager
  + GNOME Boxes
- Flathub
- Tailscale
- Automatic software updates/garbage collection
- Steam (only on Desktop)
- Nvidia drivers (only/required on Desktop)

*** Quirks
- Don't use the GNOME store (very slow and janky)
  + Install Bazaar on Flathub as a replacement
- Enroll your fingerprint through =fprintd-enroll=
  + GUI is not picking it up for some reason
- Log into Tailscale using =sudo tailscale login=

** Configuration
Fill out the =my= attribute set with your information.
=stateVersion= corresponds to the =system.stateVersion= in your original =configuration.nix=.
=dotfiles= refers to the place where you cloned this repository.
Uncomment the laptop or desktop import if you want specific customizations.

*** Laptop
Provides support for my Framework Laptop 13 with the Ryzen 5 7640U (via the NixOS hardware channels).
I also remap some keys to my liking:

- =leftalt= <-> =leftmeta=
- =capslock= <-> =esc=

*** Desktop
Provides support for my gaming PC with a RTX 4070 Super (Nvidia drivers).
I also install Steam with GE-Proton support.
