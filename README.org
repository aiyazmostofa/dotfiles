* Dotfiles
My dotfiles.

** Usage
Tested on Bluefin DX and Bazzite DX. Run the =setup.sh= script
initially. Run the =sync.sh= to sync.

If you are not using a UBlue machine, then ensure that you have =brew= and =fish= installed first.
